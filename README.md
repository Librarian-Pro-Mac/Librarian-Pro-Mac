## Overview

Librarian Pro Professional Suite is a complete personal inventory management environment that handles the full lifecycle of a collection from initial cataloging through condition tracking, valuation, and lending management.

Cross-platform database compatibility allows the same library file to be used on both Mac and Windows, making it practical for households with mixed computing environments to maintain a single shared collection database.

Export capabilities produce web pages, printable reports, and various file formats from your collection data, enabling sharing of your catalog with others or archiving it in formats accessible outside the application.

Multi-database support manages separate inventories for different collection types or different family members, keeping each collection organized and independently queryable without merging unrelated items into a single undifferentiated list.

Librarian Pro Professional Suite is the collection management platform for serious collectors and media enthusiasts who want their personal library to be as well-organized and searchable as a professional inventory system.

## Performance & Integration

Librarian Pro is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
